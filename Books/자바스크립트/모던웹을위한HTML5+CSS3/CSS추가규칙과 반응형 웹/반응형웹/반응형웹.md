| 속성                | 설명                                |
| ------------------- | ----------------------------------- |
| width               | 화면의 너비                         |
| height              | 화면의 높이                         |
| device-width        | 장치의 너비                         |
| device-height       | 장치의 높이                         |
| orientation         | 장치의 방향                         |
| device-aspect-ratio | 화면의 비율                         |
| color               | 장치의 색상 비트                    |
| color-index         | 장치에서 표현 가능한 최대 색상 개수 |
| monochrome          | 흑백 장치의 픽셀당 비트 수          |
| resolution          | 장치의 해상도                       |
